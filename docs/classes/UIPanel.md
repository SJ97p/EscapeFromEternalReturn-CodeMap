# UIPanel

## Role

인벤토리, 장비창, 창고, 제작대 등 모든 UI 패널이 따르는 공통 패널 규격입니다.

## Responsibilities

- 패널 ID 보유
- 열림/닫힘 상태 처리
- UI Manager가 동일한 방식으로 제어할 수 있는 기반 제공
- 새 패널 추가 시 공통 제어 구조 유지
