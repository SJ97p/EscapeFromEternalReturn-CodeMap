# MonsterController

## Role

일반 몬스터의 이동, 전투, 감지, 애니메이션 등 런타임 컴포넌트를 묶는 컨트롤러입니다.

## Responsibilities

- 몬스터 데이터와 런타임 컴포넌트 연결
- 전투/이동/감지 흐름의 상위 진입점 제공
- 상태 변화에 따라 애니메이션과 UI 갱신 연결
- 스폰 시스템과 몬스터 인스턴스의 접점 역할
