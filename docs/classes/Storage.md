# Storage

## Role

보관함/루팅 컨테이너 계열의 슬롯 데이터를 관리하는 클래스입니다.

## Responsibilities

- 컨테이너 슬롯 데이터 보유
- 슬롯별 아이템 추가/제거
- 보관함 UI와 Adapter 계층에 데이터 제공
- 루팅/퀵무브/드래그 이동의 대상 컨테이너 역할 수행
